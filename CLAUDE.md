## Project Overview

NDISSDAAutomate is a Python project related to NDIS (National Disability Insurance Scheme) Specialist Disability Accommodation (SDA) automation.

## Environment

- **Python**: 3.14 (virtual environment in `.venv/`)
- **IDE**: PyCharm

## Setup

```bash
source .venv/bin/activate   # or .venv\Scripts\activate on Windows CMD
pip install -r requirements.txt
```
